Fullscale

Fullscale is a GSAP exploration project of Christian Penales. This project is a compiled explorations of different Web Design and Development + GSAP Animations. This project uses HTML, CSS, and JavaScript + Vite + GSAP + Lenis. Most of the inspirations are from AWWWARDS websites and are recreated by means of creative exploration.
